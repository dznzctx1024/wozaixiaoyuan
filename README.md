# 我在校园健康打卡

## 使用说明
1.补全必要的参数(用户名，密码，打卡内容，推送的机器人id等)。  
2.定时运行即可(比如青龙面板)！
## 废话
随手写的应付学校最近的打卡，本以为XG过去就用不上了，最近流感多发学校又让搞上了，鉴于本人早上实在是起不来，参照前人所写，自己写了个“写死”自用的程序，扔进青龙面板定时运行。
